

# 设计 APP 图标（1024x1024 PNG）

## 设计思路

基于项目"华人生活圈"的定位——以地图为核心、连接附近华人服务的平台，图标将包含：

- **核心元素**：地图定位标记 + 汉字"附"（取"附近"之意，与现有 favicon 保持品牌一致）
- **配色**：使用深蓝色系（项目主色 `hsl(222, 47%, 11%)`）搭配明亮渐变，现代感强
- **风格**：iOS/Android 标准圆角矩形，简洁扁平，高辨识度
- **尺寸**：1024×1024px，适用于 App Store / Google Play 上架

## 实现方式

使用 Python + Pillow 库程序化生成图标：
1. 绘制渐变背景（深蓝到亮蓝）
2. 绘制白色地图标记图形
3. 在标记中央放置"附"字
4. 输出为 `/mnt/documents/app-icon.png`

## 输出

一个 1024×1024 的 PNG 文件，可直接用于 iOS/Android 应用商店。
